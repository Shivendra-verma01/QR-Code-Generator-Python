Custom QR Code Generator using Python

A simple Python project that generates QR codes from user input (text or URL) with color customization and image saving functionality.

Features:
1. Generate QR codes from text or URLs
2.Custom QR color and background color
3.Save QR code as PNG image
4.Input validation for empty data
5.Basic error handling for invalid colors
6.Displays generation time

Technologies Used:
1.Python
2.qrcode library
3.Pillow (PIL)

How to Run:
1. Step 1: Install required libraries:
   pip install qrcode[pil]

2. Step 2: Run the program:
   python qr_generator.py
